How to Load Form Genie Extension

Step 1: Open Chrome Extensions Page

Open Chrome and navigate to chrome://extensions/ in the address bar.

Step 2: Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the page.

Step 3: Load the Extension

Click the "Load unpacked" button that appears after enabling Developer mode.

Navigate to the Form Genie directory and select it.

Important: Make sure you select the directory that contains the manifest.json file.

Step 4: Verify Installation

The Form Genie extension should now appear in your list of extensions.

You should see the Form Genie icon in your Chrome toolbar.

Troubleshooting

If you see an error like "Manifest file is missing or unreadable":

  • Make sure you're selecting the correct directory that contains the manifest.json file.
  • Check that the manifest.json file is not corrupted or empty.
  • Ensure that all the files referenced in the manifest exist in the correct locations.
